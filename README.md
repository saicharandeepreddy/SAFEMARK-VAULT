SAFEMARK-VAULT
SAFEMARK-VAULT is a secure, lightweight credential and data management system built with Python. It leverages a local SQLite ledger to store information, ensuring your data stays under your control while providing a programmatic interface for management.

🚀 Features
Local Storage: Uses safemark_ledger.db for local-first data persistence.

Python Powered: Built with a modern Python stack for reliability and speed.

Dependency Management: Utilizes Poetry for clean and reproducible environment setups.

Extensible: Simple architecture allows for easy integration of encryption or additional metadata fields.
