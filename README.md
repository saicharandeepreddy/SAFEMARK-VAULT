SAFEMARK-VAULTSAFEMARK-VAULT is a secure, lightweight credential and data management system built with Python. It leverages a local SQLite ledger to store information, ensuring your data stays under your control while providing a programmatic interface for management.🚀 FeaturesLocal Storage: Uses safemark_ledger.db for local-first data persistence.Python Powered: Built with a modern Python stack for reliability and speed.Dependency Management: Utilizes Poetry for clean and reproducible environment setups.Extensible: Simple architecture allows for easy integration of encryption or additional metadata fields.🛠️ Installation & SetupPrerequisitesPython 3.10 or higherPoetry (recommended)1. Clone the RepositoryBashgit clone https://github.com/saicharandeepreddy/SAFEMARK-VAULT.git
cd SAFEMARK-VAULT
2. Install DependenciesIf you have Poetry installed:Bashpoetry install
Alternatively, if you prefer standard pip:Bashpip install .
💻 UsageTo launch the main application:Bashpython main.py
Note: On the first run, the system will initialize the safemark_ledger.db file if it doesn't already exist.📂 Project StructureFileDescriptionmain.pyThe entry point of the application.safemark_ledger.dbThe SQLite database storing your vault entries.pyproject.tomlProject metadata and dependency definitions.poetry.lockLocked versions of dependencies for environment consistency..replitConfiguration for running the project on Replit.🔒 Security NoteWhile SAFEMARK-VAULT provides a structured way to store data, always ensure that your local machine is secure.
